## Current Features
- Web interface built with Streamlit
- Resume and job description input side-by-side
- Text preprocessing using Pandas:
  - Lowercasing
  - Punctuation and number removal
  - Tokenization
  - Short-word filtering
- Cleaned tokens displayed as Pandas DataFrames

## Tech Stack
- Python
- Streamlit
- Pandas
- Regular Expressions
- Git

## How to Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
